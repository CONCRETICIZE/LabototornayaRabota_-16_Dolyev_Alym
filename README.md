ЛАБОРАТОРНАЯ РАБОТА 16. РАБОТА С GITHUB.


ЧТО СЛЕДУЕТ СДЕЛАТЬ:

1. Зайти в свою учётную запись на гитхабе.
2. Создать новый репозиторий.
3. Скачаем гитхаб-клиент.
4. Создадим в вебшторме новый проект.
5. Создадим токен.
6. Подружим вебшторм с гитхаб-клиентом.
7. Закоммитим его первый раз.
7.1. Изменим Readme.
7.2. Инициализируем.
7.3. Добавим Readme для коммитов.
7.4. Пометим коммит.
7.5. Выберем ветку.
7.6. Настроим путь к репозиторию.
7.7. Запушим файлы.
8. Внесём изменения в проект и закоммитим их.
9. Опять проводим изменения (удаляем файлы) и коммитим.
10. Изменяем файл и коммитим.
11. Добавляем файл и коммитим.
